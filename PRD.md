# Vinci - Product Requirements

**Problem**: New parents face cognitive impairment from sleep deprivation. The critical failure point is shift handoff - without accurate data transfer, parents either wake the sleeping partner or make educated guesses (potentially harmful).

**Solution**: Eliminate information asymmetry between co-parents during shift handoffs with a one-thumb, one-glance logging app.

---

## User Stories

| Priority | Story |
|----------|-------|
| P0 | Log a feeding/diaper/sleep event with one thumb in <10 seconds |
| P0 | See new events appear instantly on partner's device |
| P0 | View unified chronological timeline of all events |
| P0 | Use app in complete darkness without being blinded |
| P1 | Track which breast was used last |
| P1 | Track pumping sessions and yields |
| P1 | Log medications and temperature |
| P2 | View daily/weekly summary (total sleep, feeding) |

---

## What's NOT Included (MVP)

- Multiple babies
- Nannies/grandparents access
- Analytics dashboards
- Export to PDF/pediatrician reports

---

## Success Criteria

- **Time to log event**: <10 seconds
- **Sync latency**: <1 second
- **Dark mode**: No parent reports being blinded at 3am
